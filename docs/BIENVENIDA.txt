🔧 QUÉ CAMBIÓ:

El archivo script.js fue reparado para reconocer correctamente:

ANTES (Roto):
  La función getFileType() rechazaba documentos 
  porque el navegador no enviaba el MIME type correcto

DESPUÉS (Reparado):
  Ahora verifica 5 niveles diferentes:
  1. Extensión del archivo (primero)
  2. MIME type oficial
  3. Palabras clave en MIME
  4. Extensión de respaldo
  5. Rechaza solo si nada coincide

RESULTADO: ✅ Acepta casi cualquier tipo de documento

❓ PREGUNTAS FRECUENTES:

P: ¿Por qué antes no funcionaba?
R: El código era muy restrictivo. Solo aceptaba archivos 
   con MIME types que el navegador reconocía. Los PDFs, 
   Word, etc., a menudo tienen MIME genéricos.

P: ¿Ahora acepta cualquier tipo?
R: Acepta documentos conocidos. Si intentas subir 
   un archivo raro sin extensión → lo rechaza.

P: ¿Se guardan en la nube?
R: No, todo se guarda LOCALMENTE en tu navegador.

P: ¿Se pierden si cierro el navegador?
R: No, se guardan permanentemente (hasta que limpies caché).

P: ¿Puedo compartir los archivos?
R: No, cada navegador tiene su propio almacenamiento.
